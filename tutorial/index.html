hello, world 🌍

the 5-minute tour of the platform

World turns a folder of static files into <name>.your-world — employee-only, instant, with a built-in database, AI, uploads, realtime and identity. One script tag, no keys. This page is itself a World site using only the public SDK.

1 · Make a site

A folder with an index.html. Add the SDK with one tag:

<script src="/world.js"></script>

2 · Know who's visiting — world.me()

const me = await world.me();   // { email, name, handle, avatar_url }
→ …

3 · Store & sync data — world.db

Named collections of JSON docs, scoped to your site, realtime across tabs. Sign the guestbook:

const book = world.db.collection("guestbook");
await book.create({ text, by: me.name });
book.subscribe(ev => render(ev.doc));   // live

    4 · Ask the model — world.ai (streaming)

    await world.ai.complete({ prompt, stream: true, onToken: t => append(t) });

    5 · Ship it

    Three equivalent ways — pick one:

    world deploy                         # the CLI, from your folder
    # or drag the folder onto your World homepage in the browser
# or give Claude the MCP server and just ask:
    Add the MCP server your-world/mcp to Claude, then say:
"Deploy this folder to World as 'my-demo'."

    More

    Full SDK: /docs · machine-readable: /llms.txt · see every site in the universe. Also: world.uploads (files), world.ws (multiplayer channels), world.notify.slack (ping a channel). No backends, no secrets, no cron — that's the deal.
